# GoDigitalPro — Business Workspace

This folder is **separate from the website codebase**. It holds all strategy docs, client files, Whop product planning, and operational resources.

## Folder Structure

```
_business/
├── strategy/          ← 6-month plan, positioning, growth roadmap
├── clients/           ← One subfolder per client (briefs, reports, comms)
├── whop-products/     ← Whop marketplace product planning & assets
├── agency-partner/    ← Agency Partner program docs, onboarding, contracts
├── operations/        ← SOPs, checklists, internal processes
└── templates/         ← Reusable templates (proposals, reports, outreach)
```

## Quick Nav

| Folder | What's inside |
|--------|--------------|
| `strategy/` | Phase 1 & 2 roadmap, ICP docs, positioning notes |
| `clients/` | Per-client folders: briefs, audits, reports, comms |
| `whop-products/` | Product ideas, pricing, landing copy, launch plan |
| `agency-partner/` | Partner program structure, pitch decks, contracts |
| `operations/` | SOPs, onboarding checklists, workflow docs |
| `templates/` | Proposal decks, email sequences, report templates |

> Website source code lives in `src/` — don't mix business docs there.
