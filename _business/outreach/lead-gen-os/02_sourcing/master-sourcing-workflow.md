# Master Sourcing Workflow

## Step 1: Pick Niche + City/Region
- Example: Real Estate + Mumbai

## Step 2: Pull Raw Leads
- Use 2-3 sources only per batch
- Target 100 raw leads per niche/week

## Step 3: Enrich Basics
Required fields:
- company_name
- website
- contact_name
- email
- niche
- location
- notes

## Step 4: Score Leads
- Use `03_qualification_scoring/lead-score-model.md`
- Only outreach leads above threshold

## Step 5: Push To Outreach CRM
- Set status = NEW
- Start automation workflow
