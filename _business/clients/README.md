# Clients

One subfolder per client. Structure each client folder as:

```
clients/
└── client-name/
    ├── brief.md          ← Business, goals, budget, timeline
    ├── audit.md          ← Initial audit findings
    ├── proposal.md       ← Sent proposal
    ├── onboarding.md     ← Onboarding checklist & notes
    ├── reports/          ← Monthly performance reports
    └── comms/            ← Key email threads, meeting notes
```
